# ravencarrigg.github.io
Personal author site for Raven Carrigg, also known as Raven Indigo Carrigg. Author of Liminal Afterhood: Fragments from the Edge of Survival. Portland-based nonbinary poet, OHA-certified Peer Support Specialist, and trauma-informed systems navigator. Experimental poetry, trauma-survival writing, queer/trans poetry, and computational poetics.
