# Raven Indigo Carrigg

This repository hosts the personal website and creative archive of Raven Indigo Carrigg, a queer poet, computational artist, creative coder, and peer support specialist based in Portland, Oregon. The site brings together poetry, creative coding, systems thinking, trauma-survival writing, queer/trans poetics, lived-experience work, and experiments with language, number, and pattern. Across these projects, Raven explores emergent forms for tracing complexity, reclaiming voice, and making meaning at the edge of chaos.

## About this site

This website serves as a living portfolio and public archive for Raven’s creative, technical, and systems-oriented work. It includes information about *Liminal Afterhood: Fragments from the Edge of Survival*, along with experimental poetry, computational poetics, language experiments, creative coding, modeling, visualization, and public project documentation.

## Featured work

- *Liminal Afterhood: Fragments from the Edge of Survival*, an experimental poetry collection and trauma-survival archive
- Experimental poetry, trauma-survival writing, and queer/trans poetics
- Computational poetics and language experiments
- Creative coding, modeling, visualization, and computational art
- Public profiles, writing, and project documentation
